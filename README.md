# GeniyIdiot
Шуточный тест ставит "диагноз" игроку. Написан для консоли и в WinForms приложении с использованием библиотеки и хранением данных в файловой системе.

## Человеческий параграф
### Правила игры
Пользователю нужно отвечать на вопросы за отведенное время

### Конец игры
Когда **пользователь** ответил на все вопросы, он получает _"диагноз"_ и может посмеяться :)

## Техническая часть
### Сериализация
• Сохранял результаты пользователей с помощью сериализации и десериализации данных в формате JSON.

• Через NuGet Gallery скачал готовый класс Console Table для того чтобы хранить в этой таблице результаты пользователей.

• Для подсчета времени использовал Timer и вместе с ним использовал ProgressBar который заполняется, на ответ дается 10 секунд.

• Реализовал общую библеотеку, которая работает на консольное и WInForm приложения.
